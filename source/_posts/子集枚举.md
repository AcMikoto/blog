常规枚举子集可以用回溯算法，缺点是代码相对更长、常数更高，因此我们可以采用位运算的方法枚举某个s（用二进制表示状态）的子集，好处在于代码非常好写，常数低，容易结合具体操作。

枚举s的子集
```
int n = s.size(), tot = (1 << n) - 1;
for(int i = tot; i; i = tot & (i - 1)) {
	//具体操作
}
//时间复杂度2^n
```

枚举s的大小为r的子集
```
int n = s.size()
for(int i = (1 << r) - 1; i < 1 << n;) {
	//具体操作
  int y = i + (i & -i);
  i = (i & ~y) >> __builtin_ctz(i) + 1 | y;
}

//时间复杂度Cn r 肥肠高效 
```
